# University Pitch Asset Checklist

Use this checklist to turn the Startup Launchpad PRD into a complete internal pitch package.

## Core Assets Already Planned

1. PRD
   - Purpose: Defines the program, target users, operating model, timeline, resources, metrics, and risks.
   - Audience: Center leadership, deans, faculty partners, advancement, student affairs.

2. Visual Diagram
   - Purpose: Makes the four-semester pathway, entry points, events, and Sunstone progression legible in one glance.
   - Format: Mermaid diagram, Figma board, or one-page PDF.

3. Structured Proposal Document
   - Purpose: Turns the PRD into an administrative proposal with rationale, outcomes, implementation plan, budget, and approval ask.
   - Format: 6-10 page PDF or DOCX.

4. 20-Slide Deck
   - Purpose: Supports a 10-15 minute stakeholder pitch.
   - Format: PPTX, Google Slides, or PDF.

5. 60-Second Video Trailer
   - Purpose: Creates emotional buy-in for students, sponsors, and university stakeholders.
   - Format: 16:9 video, captioned, with version for web/social.

6. FAQ
   - Purpose: Answers predictable questions from students, faculty, administrators, mentors, judges, and sponsors.
   - Format: Web page or PDF.

7. Required Resources Manifest
   - Purpose: Names the staffing, space, budget, mentor, tool, and event resources required to run the program.
   - Format: Table with essential, recommended, and stretch resources.

8. Calendar and Timeline
   - Purpose: Shows the annual rhythm from recruitment through Sunstone qualification.
   - Format: Academic-year calendar, Gantt-style timeline, and event milestone view.

## Additional Documents and Assets to Pitch to the University

### Decision-Maker Package

- One-page executive summary.
- Formal approval memo with specific asks.
- Pilot-year budget.
- Staffing model with roles and estimated hours.
- Risk and compliance memo.
- Assessment plan tied to university learning outcomes.

### Student-Facing Package

- Student recruitment one-pager.
- Program landing page copy.
- Application or intake form.
- Team formation survey.
- Semester-by-semester student roadmap.
- Badge/certificate language if recognition is offered.

### Faculty and Mentor Package

- Faculty partner invitation.
- Mentor role description.
- Mentor onboarding guide.
- Office-hours calendar template.
- Judge rubric and scoring form.
- Feedback form template.

### Sponsor and Advancement Package

- Sponsorship prospectus.
- Named-prize opportunities.
- Mentor/sponsor recognition plan.
- Donor impact report template.
- External partner invitation letter.

### Operating Package

- Event run-of-show for Ideathon.
- Event run-of-show for Innovation Festival.
- Sunstone readiness checklist.
- Launchpad tool onboarding guide.
- Communications plan.
- Data tracking dashboard template.
- Post-event retrospective template.

### Evidence Package

- Benchmark examples from Stanford Longevity Design Challenge, NASA Space Apps, Hult Prize, VentureWell E-Team, MIT $100K, Rice Business Plan Competition, and Sunstone.
- Sample student journey cards.
- Sample pitch deck from a hypothetical student team.
- Before/after example showing how Launchpad improves an idea.
- Success metrics dashboard mockup.
