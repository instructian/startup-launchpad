# Codex Prompt Bundle: Startup Launchpad Pitch Assets

Use these prompts with Codex to generate polished files in a local workspace. Each prompt assumes Codex can create Markdown, Mermaid, HTML, PPTX, scripts, or structured project files as appropriate.

## Shared Context for Every Prompt

Create assets for "Startup Launchpad," a four-semester extracurricular university innovation program. The program helps students develop a startup mindset by working in small cross-functional teams through repeated cycles of idea discovery, validation, prototyping, pitching, feedback, and iteration.

Primary entry points:

- Fall Year 1 for full-pathway students.
- Fall Year 2 for students with an existing idea, prototype, capstone, research project, creative product, or early traction.
- Rolling entry for students joining teams as contributors.

Annual milestones:

- Fall Ideathon.
- Spring Innovation Festival.
- Advanced Sunstone competition pathway.

The Launchpad tool supports idea framing, validation, evidence tracking, prototype planning, team role mapping, pitch preparation, and Sunstone readiness.

## 1. Visual Diagram Prompt

Create a polished visual diagram package for Startup Launchpad.

Requirements:

- Create `outputs/startup-launchpad-diagram.md` with a Mermaid diagram.
- Create `outputs/startup-launchpad-diagram.html` with a presentation-ready visual version.
- Show the four-semester path, flexible entry tracks, event ladder, Launchpad tool support layer, and core loop.
- Include brief alt text and a caption suitable for a university proposal.
- Keep styling professional, readable, and export-friendly.

## 2. Structured Proposal Document Prompt

Create a structured proposal document for university stakeholders.

Requirements:

- Create `outputs/startup-launchpad-proposal.md`.
- Organize it as a formal proposal with executive summary, rationale, program model, timeline, resources, budget categories, assessment metrics, risks, and approval ask.
- Include tables where useful.
- Include a one-page executive summary at the top.
- Write in a credible internal university voice.

## 3. 20-Slide Deck Prompt

Create a 20-slide presentation deck for Startup Launchpad.

Requirements:

- Create `outputs/startup-launchpad-20-slide-deck.md` with slide-by-slide content and speaker notes.
- If the workspace supports PPTX generation, also create `outputs/startup-launchpad-20-slide-deck.pptx`.
- Include exactly 20 slides.
- Make the deck suitable for a 10-15 minute university stakeholder pitch.
- Use concise slide text, strong story progression, and notes with the fuller speaking argument.

Slide sequence:

1. Title
2. Student innovation gap
3. Why now
4. Program vision
5. Target students
6. Core loop
7. Four-semester roadmap
8. Flexible entry model
9. Team formation model
10. Launchpad tool role
11. Fall Ideathon
12. Spring Innovation Festival
13. Sunstone pathway
14. External challenge benchmarks
15. Resource model
16. Pilot-year calendar
17. Success metrics
18. Risks and mitigations
19. University approval ask
20. Closing vision

## 4. 60-Second Video Trailer Prompt

Create a production-ready 60-second video trailer package for Startup Launchpad.

Requirements:

- Create `outputs/startup-launchpad-video-trailer.md`.
- Include timestamped script, voiceover, scene descriptions, on-screen text, b-roll ideas, music direction, pacing, and caption text.
- If using Remotion or another video framework, create a minimal renderable project and document how to render it.
- The trailer should show a student journey from idea to team to prototype to pitch to Sunstone readiness.

## 5. FAQ Prompt

Create a comprehensive FAQ for Startup Launchpad.

Requirements:

- Create `outputs/startup-launchpad-faq.md`.
- Include sections for students, teams, faculty, mentors, judges, sponsors, and administrators.
- Address entry timing, whether students need an idea, team formation, events, Launchpad tool use, Sunstone qualification, expected time commitment, resources, judging, equity/access, and outcomes.
- Keep answers direct and reassuring.

## 6. Required Resources Manifest Prompt

Create a required resources manifest for Startup Launchpad.

Requirements:

- Create `outputs/startup-launchpad-resources-manifest.md`.
- Include staffing, budget, spaces, mentors, judges, workshops, technology, Launchpad tool support, marketing, event production, prizes, and microgrants.
- Classify every resource as essential, recommended, or stretch.
- Include owner, timing, dependency, risk if missing, and suggested minimum viable pilot version.

## 7. Calendar and Timeline Prompt

Create a two-year calendar and timeline for Startup Launchpad.

Requirements:

- Create `outputs/startup-launchpad-calendar-timeline.md`.
- Include a semester-by-semester calendar and a month-by-month operating timeline.
- Include recruitment, intake, team formation, workshops, Launchpad module deadlines, mentor office hours, Ideathon, Innovation Festival, Sunstone prep, judging, and summer pilot sprint.
- Include a compact visual timeline in Mermaid.

## 8. Pitch Asset Recommendation Prompt

Create a recommendation memo listing additional documents and assets needed to pitch Startup Launchpad to university leadership.

Requirements:

- Create `outputs/startup-launchpad-pitch-assets-memo.md`.
- Group recommendations by audience: decision-makers, students, faculty, mentors, sponsors, operations, and assessment.
- For each asset, include purpose, format, owner, and when it is needed.
- End with a minimum viable pitch package and an ideal full package.
