# Replit Prompt Bundle: Startup Launchpad Pitch Assets

Use these prompts in Replit to generate web-native, visual, or exportable assets for the Startup Launchpad university pitch package.

## Shared Context for Every Prompt

You are creating assets for "Startup Launchpad," a four-semester extracurricular university program that helps students develop a startup and innovation mindset. Students enter primarily in Fall Year 1 or Fall Year 2, with rolling entry for team contributors. The program uses a fall Ideathon, spring Innovation Festival, and advanced Sunstone competition. The Launchpad tool helps students frame ideas, validate customers, build prototypes, track evidence, and prepare increasingly demanding pitches.

Core loop: Provocation -> Team Formation -> Customer Evidence -> Build/Test -> Pitch -> Judge Feedback -> Next Sprint.

Tone: credible, energetic, university-ready, student-centered, practical, not hype-heavy.

## 1. Visual Diagram Prompt

Create a responsive web page that visualizes the Startup Launchpad pathway. Use HTML, CSS, and JavaScript or React. The diagram must show:

- Four-semester progression: Fall Y1 Discover, Spring Y1 Validate, Fall Y2 Build, Spring Y2 Launch.
- Entry tracks: Explorer, Joiner, Accelerator, Contributor.
- Events: Ideathon, Innovation Festival, Advanced Ideathon, Sunstone Qualification.
- Launchpad tool as a support layer across all stages.
- Core loop: Provocation, Team Formation, Customer Evidence, Build/Test, Pitch, Feedback, Next Sprint.

Deliverables:

- A polished visual diagram page.
- Mobile and desktop responsive layouts.
- Export-friendly screenshot area.
- Clear labels and professional university presentation styling.

## 2. Structured Proposal Document Prompt

Create a structured proposal as an HTML document that can be exported to PDF. Include:

- Executive summary.
- Problem and opportunity.
- Program model.
- Four-semester roadmap.
- Student entry tracks.
- Event ladder.
- Required resources.
- Implementation timeline.
- Assessment metrics.
- University approval ask.

Style it as a professional internal university proposal, not a marketing landing page. Include tables, callout boxes, and a one-page executive summary at the top.

## 3. 20-Slide Deck Prompt

Build a browser-based slide deck for Startup Launchpad using Reveal.js or a similar lightweight web presentation library. Create exactly 20 slides:

1. Title
2. The student innovation gap
3. Why now
4. Program vision
5. Target students
6. Core loop
7. Four-semester roadmap
8. Flexible entry model
9. Team formation model
10. Launchpad tool role
11. Fall Ideathon
12. Spring Innovation Festival
13. Sunstone pathway
14. External challenge benchmarks
15. Resource model
16. Pilot-year calendar
17. Success metrics
18. Risks and mitigations
19. University approval ask
20. Closing vision

Use strong hierarchy, restrained colors, clean diagrams, and presenter notes for each slide.

## 4. 60-Second Video Trailer Prompt

Create a short web-based storyboard and production package for a 60-second trailer introducing Startup Launchpad. Output:

- A 60-second script with timestamps.
- Scene-by-scene storyboard cards.
- Suggested visuals for each scene.
- On-screen text.
- Voiceover.
- Music and pacing notes.
- A simple animated HTML preview if feasible.

The trailer should make the program feel active, student-centered, and credible. It should show students moving from idea to team to prototype to pitch to Sunstone readiness.

## 5. FAQ Prompt

Create a responsive FAQ web page for Startup Launchpad with categories:

- Students
- Teams
- Faculty
- Mentors
- Judges
- Sponsors
- University administrators

Answer questions about who can join, when students can enter, whether they need an idea, how teams form, how Launchpad is used, what the events are, how Sunstone qualification works, what resources are available, and what outcomes the university should expect.

## 6. Required Resources Manifest Prompt

Create an interactive resources manifest for Startup Launchpad. It should include:

- Staffing
- Budget
- Spaces
- Mentors
- Judges
- Workshops
- Technology
- Launchpad tool support
- Marketing and recruitment
- Event production
- Microgrants and prizes

Classify each resource as essential, recommended, or stretch. Include estimated ownership, timing, dependencies, and risk if missing.

## 7. Calendar and Timeline Prompt

Create an academic-year calendar and two-year timeline for Startup Launchpad. Include:

- Recruitment windows.
- Team formation.
- Workshops.
- Launchpad module deadlines.
- Fall Ideathon.
- Spring Innovation Festival.
- Sunstone readiness activities.
- Mentor office hours.
- Application and judging deadlines.
- Summer optional pilot sprint.

Create both a monthly calendar view and a horizontal timeline view.
